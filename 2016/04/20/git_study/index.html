解决了。

用命令 git commit告诉Git，把文件提交到仓库。

现在我们已经提交了一个readme.txt文件了，我们下面可以通过命令git status来查看是否还有文件未提交

git status

On branch master
Your branch is ahead of ‘origin/master’ by 1 commits.
(use “git push” to publish your local commits)
nothing to commit, working directory clean

没有可提交文件。

文件删除
git rm 文件名

示例：

git rm fifle

如果删除错了文件。怎么回归到上一个版本。

git reset –hard 版本号

如何查询版本号

查询版本
git reflog

ac4f465 HEAD@{0}: commit: delete fifle
ca1ee69 HEAD@{1}: commit: test commit
400552d HEAD@{2}: commit: add file
037ad92 HEAD@{3}: commit: init commit
c06935c HEAD@{4}: reset: moving to c06935c
28d4430 HEAD@{5}: reset: moving to HEAD^
c06935c HEAD@{6}: commit: add AAAAAA
28d4430 HEAD@{7}: commit: add chenge
48682d0 HEAD@{8}: commit (initial): readme.txt

选择你要回归的版本

git reset – ca1ee69

HEAD is now at ca1ee69 test commit

上一个版本的东西就回来啦。。

试试提交到github

首先，登录github，创建一个新的仓库，名字叫testgit

新建完后
github 提示

echo “# testgit” >> README.md
git init
git add README.md
git commit -m “first commit”
git remote add origin https://github.com/zhangkangbin/testgit.git
git push -u origin master

下面这两句就是提交到上面去

输入账号和密码（也可以用ssh key 免登陆）
